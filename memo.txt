git status ~~~：変更が加えられたけどステージングされていないもの or ステージングに登録されているけどコミットされていない変更が確認できる
git add ~~~：ステージングに登録する
git commit -m "○○"：○○という名前でコミットする
git log：コミットされたログを確認することができる。確認項目はコミットのタイトルとユーザーと更新日時とコミットされたID
git show：変更の詳細が見れる。
git show なにかしらのID：過去に更新したコミットIDの詳細を見ることができる
git diff：ステージング前の変更が加えられたファイルがどのように変更を加えられたのか見ることができる
git diff --cached：ステージングされているけどコミットされていない変更が加えられているファイルの変更箇所を見ることができる
